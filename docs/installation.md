# Installation

Requirements
- Minecraft 1.21.11
- Fabric Loader 0.19.2+ (use Fabric Installer)
- Java 17+

Client install
1. Download the latest release JAR from the project's Releases page.
2. Place the JAR in your `.minecraft/mods` folder.
3. Launch Minecraft using the Fabric Loader profile.

Development build
1. Clone the repository:

```bash
git clone https://github.com/Falthera/AnchorLikeMarlow.git
cd AnchorLikeMarlow
```

2. Build with Gradle:

```bash
./gradlew build
# Output jar is in build/libs/
```

3. Copy the built JAR to `.minecraft/mods` for local testing.
