# AnchorLikeMarlow (ALM)

AnchorLikeMarlow is a lightweight, performance-focused Fabric client mod that assists with safe respawn-anchor interactions. This fork prioritizes runtime efficiency: no HUD, no keybinds, no commands — only fast, reliable assist logic that minimizes allocations and CPU on the client.

Core goals:
- Minimal allocations and CPU usage
- No UI overhead (HUD disabled by default)
- Safe synthetic dispatches guarded by `PacketGuard`
